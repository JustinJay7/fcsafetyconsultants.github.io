# FC Safety Consultants – My Website

Hi, I built this website using **Lovable**. I designed it to help businesses find reliable health and safety consulting services. After building it in Lovable, I downloaded the code as a ZIP file and deployed it on **GitHub Pages** so it runs for free and stays live.

This website is fully functional, responsive, and ready for real-world use.

## What This Website Can Do

Here are the main features I built into this site:

- **Show safety consulting services** – Visitors can see all the safety services I offer, including risk assessments, compliance audits, and safety training.
- **Contact form** – Clients can fill out a form to request a quote or ask questions. Messages go directly to me.
- **Client login area** – Existing clients can log in to access reports, safety plans, and documents.
- **Resource library** – I added free safety guides and checklists that anyone can download.
- **News and updates section** – I post the latest safety regulations and company news here.
- **Fully responsive design** – The website works perfectly on phones, tablets, and computers.
- **Fast loading** – I optimized everything so pages load quickly even on slow internet.
- **SEO ready** – The site is structured to appear on Google when people search for safety consultants.
- **Secure HTTPS** – The site runs on a secure connection by default through GitHub Pages.

## How I Built and Deployed It

1. I built the website using **Lovable**.
2. I downloaded the full project code as a **ZIP file**.
3. I created a **public GitHub repository** named `fcsafety-website`.
4. I uploaded all the code files to that repository.
5. I enabled **GitHub Pages** in the repository settings.
6. GitHub Pages automatically deployed the site for free.
